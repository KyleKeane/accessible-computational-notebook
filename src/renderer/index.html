      Keyboard shortcuts
      
    
      Every command is also available in the menu bar (press Alt or F10).
      

    
      Navigating
      
    
      
      
      Up / Down arrow
      Previous / next cell (when the cell, not its editor, has focus)
      
      
      Enter
      Move from the cell into its editor
      
      
      Escape
      Leave the editor and focus the cell (screen reader users: leave focus/forms mode first)
      
      
      F6
      Cycle between toolbar, cells, and status bar
      
      
      Ctrl+Shift+Home / End
      First / last cell
      
      
      Ctrl+Shift+J
      Go to a section (markdown headings outline)
      
      
      Ctrl+Shift+T
      Collapse or expand the section at the current heading cell
      
      
      Ctrl+Shift+E
      Read the notebook as one narrative: prose, code steps, and results in order
      
      
      Ctrl+Shift+N
      Describe the notebook (name, cells, sections, kernel)
      
      
      Ctrl+Shift+L
      Review the announcement history
      
      
      F3
      Repeat the last search
      
      
      Ctrl+Shift+I
      Describe the current cell
      
      
      Ctrl+Shift+O
      Read the current cell's output
      
      
      Ctrl+Shift+G
      Add or edit a description for an image output
      
      
      Ctrl+Space
      Complete the name at the cursor (kernel-aware)
      
      
      Ctrl+Shift+U
      Describe the symbol at the cursor (signature and documentation)
      
      
      Ctrl+Shift+W
      List the kernel's variables with types and values
      
      
      Ctrl+Shift+K
      Announce kernel status
      
      
      Ctrl+,
      Settings (time limits, announcement length, autosave)
      
    
      

    
      Running
      
    
      
      
      Shift+Enter
      Run cell and advance (creates a new cell at the end)
      
      
      Ctrl+Enter
      Run cell in place
      
      
      F9
      Run only the selected text; the result is spoken, nothing is stored
      
      
      Ctrl+E
      Quick evaluate: type an expression, hear its result, insert it as a cell when it's right
      
      
      Ctrl+Shift+Enter
      Run all cells
      
      
      _ and Out[n]
      In code: the last result, and the result of execution n
      
      
      Ctrl+.
      Interrupt kernel
      
      
      Ctrl+Shift+.
      Restart kernel
      
    
      

    
      Editing cells
      
    
      
      
      Ctrl+Shift+B
      Insert code cell below
      
      
      Ctrl+Shift+A
      Insert code cell above
      
      
      Ctrl+Shift+M
      Insert markdown cell below
      
      
      Ctrl+Shift+D
      Delete cell
      
      
      Ctrl+Shift+X / C / V
      Cut / copy / paste cell
      
      
      Ctrl+Shift+Minus
      Split the cell at the cursor
      
      
      Ctrl+Alt+I
      Mark or unmark an initialization cell (Cell menu runs them all)
      
      
      ?name in a cell
      Running a cell containing just ?name answers with its documentation
      
      
      Alt+Up / Alt+Down
      Move cell up / down
      
      
      Ctrl+Alt+Z / Ctrl+Alt+Y
      Undo / redo cell operations (insert, delete, move, type change)
      
      
      Ctrl+F
      Find and replace across all cells
      
      
      Tab / Shift+Tab in an editor
      Indent / unindent (press Escape first to move focus instead)
